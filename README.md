# 🍽️ Rumah Makan Medan Baru - Landing Page

Landing page restoran Padang legendaris berbasis HTML5 dan Tailwind CSS. Dioptimalkan penuh untuk **Conversion Rate Optimization (CRO)**, **Local SEO**, dan **Mobile-First UX** guna memaksimalkan reservasi meja dan kunjungan langsung (walk-in).

## 🚀 Fitur Utama

- **Mobile-First UX:** *Sticky bottom navigation* khusus untuk layar sentuh agar tombol "Menu", "Lokasi", dan "Reservasi WA" selalu terjangkau jempol pengguna.
- **Frictionless Booking:** Tombol CTA terintegrasi langsung dengan API WhatsApp beserta pesan otomatis (*pre-filled message*).
- **Trust & Social Proof:** *Trust bar* ulasan Google (Rating 4.5 dari 5.800+ ulasan) yang diletakkan strategis di bawah *Hero Section*.
- **Appetite-Inducing UI:** Palet warna psikologi makanan (Merah Cabai & Emas Gulai) dipadukan dengan tipografi premium (*Playfair Display*).
- **Local SEO Ready:** Struktur *heading* semantik (H1, H2, H3) yang dioptimalkan untuk mesin pencari.

## 🛠️ Tech Stack

- **HTML5:** Struktur dokumen semantik.
- **Tailwind CSS (via CDN):** Kerangka kerja CSS utilitas untuk *styling* yang sangat cepat dan responsif.
- **Vanilla JavaScript:** Skrip ringan untuk interaksi *toggle* menu pada versi mobile.
- **Google Fonts & FontAwesome:** Dukungan tipografi dan ikonografi modern.
